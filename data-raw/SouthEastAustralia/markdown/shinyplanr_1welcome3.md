How to use the Shiny App
-------------------

This guide gives a quick overview of how to run a planning scenario and review results. For more details, see Help / Technical Information.

### 1. Go to the Scenario tab ###

This is where you set up and run a scenario.
- **Set targets:** Choose the percentage of each feature to protect (e.g., a 20% target means 20% of that feature's area will be included in the solution).
- **Select cost layer:** Pick a cost layer (e.g., fishing, aquaculture, shipping). The tool will then try to avoid high-cost areas when meeting your targets.
- **Add climate resilience:** Make the plan climate-smart by prioritizing areas less affected by future climate change (climate refugia). The tool balances this with biodiversity objectives and minimizing conflicts with high-use areas.
- **Run analysis:** Click Run analysis. The app will generate a plan in seconds (longer if climate options are included). The solution is calculated using an optimization algorithm that finds the best balance between conservation goals and costs.

### 2. Examine the results ###

Explore details in tabs:

- **The map:** Shows which areas are selected for protection and which remain available for other uses (e.g., fishing).
- **Targets:** See how much of each feature is protected compared to your target.
- **Costs:** Check how well the plan avoids high-use areas.
- **Climate:** Review the climate-resilience score of selected areas.
- **Details:** Explore additional information about the scenario.

Information from each tab can be downloaded.

We provide a quick introduction here to run a planning scenario and to examine the results. See Help/Technical Information for more details.  

### Other Tabs in the App

We have included additional tabs to help you explore and understand each planning scenario. More details are provided within each tab.

- **Comparison:** Compare two spatial plans side-by-side.
- **Layer Information:** View details of each feature, climate, and cost layer used in the analysis.
- **Help:** Access FAQs, references and technical information.
- **Credits:** Acknowledges contributors to this Shiny App.
