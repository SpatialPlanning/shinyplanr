Welcome to shinyplanr — an interactive tool for marine spatial planning (MSP). This application allows you to explore conservation planning scenarios, set protection targets for habitats and species, and visualize the trade-offs between conservation goals and human uses of the ocean.

Shinyplanr is built on the [prioritizr](https://prioritizr.net) R package, which uses optimization algorithms to find efficient spatial plans that meet your conservation targets while minimizing impacts on existing ocean uses. The tool is designed for stakeholder engagement, allowing users to interactively adjust parameters and immediately see the results.

### Getting Started

Use the tabs at the top of the page to navigate between different sections of the app:

- **Scenario**: Build and run planning scenarios by setting targets and cost layers
- **Comparison**: Compare two different planning scenarios side-by-side
- **Layer Information**: Explore the data layers used in the analysis
- **Help**: Find answers to common questions and technical details
