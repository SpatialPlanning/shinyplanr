# Articles

### User Manual

- [Introduction to Spatial
  Planning](https://spatialplanning.github.io/shinyplanr/articles/aa-introduction.md):

  Conceptual foundation for spatial prioritisation — planning units,
  features, targets, objectives, climate-smart planning, multiple-use
  planning, and ecosystem services.

- [Using the shinyplanr
  Application](https://spatialplanning.github.io/shinyplanr/articles/ab-using-shinyplanr.md):

  A guide to each tab of the shinyplanr web application — setting
  targets, selecting cost layers, running analyses, comparing scenarios,
  and interpreting results.

- [Setting Up shinyplanr for Your
  Region](https://spatialplanning.github.io/shinyplanr/articles/ac-setting-up.md):

  Step-by-step guide for deployers: creating a deployment project with
  create_shinyplanr_template(), preparing spatial data, configuring app
  options, defining the feature dictionary, and testing locally.

- [Deploying
  shinyplanr](https://spatialplanning.github.io/shinyplanr/articles/ad-deployment.md):

  How to deploy a shinyplanr application to Posit Connect — locking
  package versions with renv, configuring rsconnect, and troubleshooting
  common deployment issues.
